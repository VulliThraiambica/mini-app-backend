#  Mini App Backend

A backend server for the Employee Management System built using Node.js, Express.js, and MongoDB Atlas.

The backend handles REST API requests, database operations, and employee data management including create, read, update, and delete functionalities.


##  Features

- REST API Development
- Create Employee API
- Read Employee API
- Update Employee API
- Delete Employee API
- MongoDB Atlas Integration
- Error Handling
- CORS Configuration
- Cloud Deployment

---

##  Tech Stack

- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- dotenv
- CORS
